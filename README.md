# Autonomous Coding Agent

وكيل برمجي مستقل قادر على أداء كل المهام التي توكل إليه بذكاء وكفاءة.

## 🚀 الميزات

### القدرات الأساسية (13 قدرة):
1. **نظام المهارات المتطور** — يُنشئ مهارات جديدة تلقائياً عند مواجهة مهام غير مألوفة
2. **بنية سرب الوكلاء** — يُنسق حتى 10 وكلاء متخصصين يعملون معاً
3. **التنفيذ المتوازي** — يُشغل المهام المستقلة بالتوازي عبر رسم بياني DAG
4. **التفكير العميق** — يُنفذ تفكير متعدد الخطوات مع تحليل المقايضات
5. **الذاكرة المستمرة** — يحتفظ بالمعرفة عبر الجلسات مع اضمحلال الأهمية
6. **الاسترداد الذاتي** — يكتشف ويتعافى من الأخطاء تلقائياً
7. **إنشاء الأدوات** — يُنشئ أدوات جديدة عند الحاجة
8. **التعلم من التغذية الراجعة** — يتحسن من تصحيحات المستخدم
9. **مراجعة الجودة** — يُراجع ويصحح كوده تلقائياً
10. **تفكيك المهام** — يُقسم المهام المعقدة إلى أجزاء قابلة للتنفيذ
11. **التحكم بالإصدارات** — يُدير التغييرات مع نقاط استرجاع
12. **فهم اللغة الطبيعية** — يفهم التعليمات بعدة لغات (العربية + الإنجليزية)
13. **مراقبة الأداء** — يُتابع ويُحسّن أداءه باستمرار

### الواجهة الرسومية (Web UI):
- 🌙 واجهة Dark Theme احترافية مع تأثيرات Glass Morphism
- 🔄 تحديثات لحظية عبر WebSocket
- 🔀 تصور DAG تفاعلي (React Flow)
- 🤖 مراقبة سرب الوكلاء مع نبضات القلب
- 📈 لوحات أداء بيانية
- 🌐 دعم كامل للعربية RTL
- 📱 تصميم متجاوب (موبايل → ديسكتوب)

## 🛠 التقنيات المستخدمة

### Backend:
- TypeScript (strict mode)
- Vitest + fast-check (Property-Based Testing)
- Event-driven Message Bus architecture

### Frontend (Web UI):
- React 18 + TypeScript
- Vite (build tool)
- Tailwind CSS (dark theme)
- Framer Motion (animations)
- React Flow (DAG visualization)
- Recharts (performance charts)
- Zustand (state management)
- i18next (internationalization)
- WebSocket (real-time)

## 📁 هيكل المشروع

```
├── src/
│   ├── shared/          # أنواع مشتركة + ناقل رسائل
│   ├── infrastructure/  # ذاكرة + تحكم إصدارات + أداء + استرداد
│   ├── execution/       # مهارات + أدوات + جودة + تغذية راجعة
│   ├── orchestration/   # تفكيك مهام + جدولة DAG + تنسيق وكلاء
│   ├── interface/       # NLU + محرك تفكير
│   ├── web-ui/          # واجهة رسومية React
│   └── index.ts         # نقطة الدخول الرئيسية
├── .kiro/specs/         # مواصفات المشروع
└── package.json
```
